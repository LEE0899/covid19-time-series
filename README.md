# COVID-19 Confirmed Cases Time Series Analysis

## Overview
- COVID-19 global confirmed cases time series analysis
- Data source: Johns Hopkins CSSE COVID-19 dataset

## What this project does
- Cleans daily report CSV files
- Unifies country names
- Aggregates confirmed cases by country
- Builds time-series data for visualization

## Key points
- Index-based merge using Country_Region
- Handling missing values in time-series data
- Exporting cleaned data for visualization tools

## Tools
- Python
- pandas
- Jupyter Notebook
